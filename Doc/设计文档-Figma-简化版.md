# 养娃不易小程序 - UI设计规范(简化版)

## 设计风格
温馨、简洁、易用,根据儿童健康状态动态切换主题色调

## 动态主题系统 ⭐核心特性

### 健康主题 (体温 < 37.3℃)
- 主题色: #7ED321 (健康绿)
- 渐变: linear-gradient(135deg, #7ED321 0%, #9BE34D 100%)
- 视觉感受: 放心、舒适、健康
- 状态标识: 🟢 健康状态良好

### 低热主题 (37.3℃ ≤ 体温 < 38.5℃)
- 主题色: #F5A623 (温暖橙)
- 渐变: linear-gradient(135deg, #F5A623 0%, #FFB84D 100%)
- 视觉感受: 关注、提醒、温暖
- 状态标识: 🟡 低热状态

### 发热主题 (体温 ≥ 38.5℃)
- 主题色: #D0021B (警示红)
- 渐变: linear-gradient(135deg, #D0021B 0%, #FF4D6A 100%)
- 视觉感受: 警示、紧急、需要关注
- 状态标识: 🔴 发热状态

### 主题切换规则
- 实时监测: 根据最新体温记录自动判断
- 平滑过渡: 主题切换使用 0.5s 渐变动画
- 全局应用: 卡片背景、按钮、状态指示器同步变化

## 色彩系统

### 主色调
- 主色: #4A90E2 (柔和蓝)
- 辅助色: #7ED321 (健康绿)
- 警示色: #F5A623 (温暖橙)
- 危险色: #D0021B (警示红)

### 功能色
- 体温正常: #7ED321
- 低热: #F5A623
- 高热: #D0021B
- 背景色: #F8F9FA
- 卡片背景: #FFFFFF
- 文字主色: #333333
- 文字辅色: #666666
- 文字弱色: #999999

## 字体规范
- 主字体: PingFang SC / 思源黑体
- 数字字体: DIN Alternate
- 大标题: 20px 加粗
- 标题: 18px 加粗
- 副标题: 16px 中等
- 正文: 14px 常规
- 辅助文字: 12px 常规
- 体温数字: 32px 加粗

## 间距与圆角
- 页面边距: 16px
- 卡片间距: 12px
- 内容间距: 8px
- 大圆角: 12px (卡片)
- 中圆角: 8px (按钮)
- 小圆角: 4px (标签)

## 核心页面

### 1. 首页
```
顶部: 儿童档案卡片(头像+姓名+年龄+过敏史+状态指示器)
测试区: 主题切换按钮(健康/低热/发热)
中部: 今日概览(体温/用药/症状三项指标)
快速记录: 体温/用药/症状三个按钮
底部: 最近记录列表(时间轴展示)
```

### 2. 体温记录页
```
顶部: 时间筛选(今天/昨天/近7天)
中部: 体温曲线图(折线图,高热标红,低热标黄)
统计: 最高/最低/平均体温
底部: 记录列表(时间轴,颜色标识状态)
```

### 3. 用药管理页
```
顶部: 用药提醒卡片(下次可用时间+今日统计)
中部: 用药记录列表(药品名+剂量+时间)
底部: 常用药品库(网格布局)
```

### 4. 添加体温记录
```
体温输入: 滑动选择器(35-42℃)
测量时间: 日期时间选择
测量部位: 单选(腋下/口腔/耳温/肛温)
备注: 文本输入
症状标签: 多选标签
```

### 5. 添加用药记录
```
药品选择: 搜索+快捷选择
药品信息: 显示用药规则
剂量: 数字输入
用药时间: 日期时间选择
用药提醒: 间隔检查警告
```

- 主按钮: 44px高, 8px圆角, 动态主题色背景, 白色文字
- 次按钮: 44px高, 8px圆角, 白色背景, 动态主题色边框
- 图标按钮: 48px×48px, 圆形, #F8F9FA背景

### 卡片
- 背景: #FFFFFF
- 圆角: 12px
- 阴影: 0 2px 8px rgba(0,0,0,0.08)
- 内边距: 16px
- 儿童档案卡片: 动态主题渐变背景

### 输入框
- 高度: 44px
- 圆角: 8px
- 边框: 1px #E0E0E0
- 聚焦边框: 1px 动态主题色

### 状态标签
- 高度: 24px
- 圆角: 12px
- 正常: #E8F5E9背景, #4CAF50文字
- 低热: #FFF3E0背景, #FF9800文字
- 高热: #FFEBEE背景, #F44336文字

## 交互设计

### 手势
- 左滑删除
- 下拉刷新
- 长按编辑

### 动画
- 页面切换: 300ms滑动
- 按钮点击: 100ms缩放
- 列表加载: 200ms渐显
- 主题切换: 500ms渐变

### 反馈
- 成功: 绿色Toast, 2秒消失
- 错误: 红色Toast, 手动关闭
- 加载: 加载动画+提示文字

## 特殊场景

### 高热警告(≥39℃)
- 体温数字红色
- 状态标签"高热"
- 弹出警告弹窗
- 显示就医建议
- 应用发热主题

### 用药间隔警告
- 显示警告弹窗
- 提示上次用药时间
- 显示建议用药时间
- 强制用药需二次确认

### 过敏提醒
- 显示红色警告
- 提示过敏史
- 建议咨询医生

## 用户体验优势

### 动态主题的价值
- **直观反馈**: 家长一眼就能看出孩子当前的健康状态
- **情感化设计**: 不同颜色传达不同的情感(放心/关注/警示)
- **减少焦虑**: 健康状态用绿色,让家长安心
- **及时提醒**: 发热状态用红色,提醒家长注意

## 响应式设计
- 最小宽度: 320px
- 最大宽度: 750px
- 使用rpx单位自适应

## 无障碍设计
- 文字背景对比度≥4.5:1
- 最小触摸区域: 44px×44px
- 按钮间距≥8px
- 支持系统字体调整
