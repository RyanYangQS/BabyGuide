# 儿童健康管理小程序 - UI设计规范

## 一、设计概述

### 1.1 产品定位
专为宝爸宝妈设计的儿童健康管理小程序,提供健康记录、用药管理和智能建议服务。

### 1.2 目标用户
- 主要用户:0-12岁儿童的家长(宝爸宝妈)
- 次要用户:其他照顾儿童的家庭成员

### 1.3 设计风格
- **整体风格**:温馨、简洁、易用
- **色彩基调**:柔和的蓝色和绿色为主,体现健康、安心
- **情感定位**:专业、贴心、可信赖
- **动态主题**:根据儿童健康状态自动切换主题色调

---

## 二、设计规范

### 2.1 色彩系统

#### 动态主题系统 ⭐核心特性
根据儿童当前体温状态,自动切换应用主题色调,提供直观的视觉反馈:

**健康主题 (体温 < 37.3℃)**
- 主题色: #7ED321 (健康绿)
- 辅助色: #9BE34D (明亮绿)
- 渐变: linear-gradient(135deg, #7ED321 0%, #9BE34D 100%)
- 应用场景: 儿童档案卡片、添加按钮、状态指示器
- 视觉感受: 放心、舒适、健康

**低热主题 (37.3℃ ≤ 体温 < 38.5℃)**
- 主题色: #F5A623 (温暖橙)
- 辅助色: #FFB84D (明亮橙)
- 渐变: linear-gradient(135deg, #F5A623 0%, #FFB84D 100%)
- 应用场景: 儿童档案卡片、添加按钮、状态指示器
- 视觉感受: 关注、提醒、温暖

**发热主题 (体温 ≥ 38.5℃)**
- 主题色: #D0021B (警示红)
- 辅助色: #FF4D6A (明亮红)
- 渐变: linear-gradient(135deg, #D0021B 0%, #FF4D6A 100%)
- 应用场景: 儿童档案卡片、添加按钮、状态指示器
- 视觉感受: 警示、紧急、需要关注

**主题切换规则**
- 实时监测: 根据最新体温记录自动判断
- 平滑过渡: 主题切换使用 0.5s 渐变动画
- 全局应用: 卡片背景、按钮、状态指示器同步变化
- 状态提示: 在儿童档案卡片显示当前状态(🟢健康/🟡低热/🔴发热)

#### 主色调
- **主色**: #4A90E2 (柔和蓝) - 用于主要按钮、重点信息
- **辅助色**: #7ED321 (健康绿) - 用于健康状态、成功提示
- **警示色**: #F5A623 (温暖橙) - 用于提醒、警告信息
- **危险色**: #D0021B (警示红) - 用于高热、紧急情况

#### 功能色
- **体温正常**: #7ED321 (绿色)
- **低热**: #F5A623 (橙色)
- **高热**: #D0021B (红色)
- **背景色**: #F8F9FA (浅灰)
- **卡片背景**: #FFFFFF (白色)
- **文字主色**: #333333
- **文字辅色**: #666666
- **文字弱色**: #999999

### 2.2 字体规范

#### 字体家族
- **主字体**: PingFang SC (iOS) / 思源黑体 (Android)
- **数字字体**: DIN Alternate (用于体温、时间等数字显示)

#### 字号规范
- **大标题**: 20px (加粗)
- **标题**: 18px (加粗)
- **副标题**: 16px (中等)
- **正文**: 14px (常规)
- **辅助文字**: 12px (常规)
- **体温数字**: 32px (加粗,DIN Alternate)

### 2.3 间距规范
- **页面边距**: 16px
- **卡片间距**: 12px
- **内容间距**: 8px
- **文字行高**: 1.5倍

### 2.4 圆角规范
- **大圆角**: 12px (卡片、弹窗)
- **中圆角**: 8px (按钮、输入框)
- **小圆角**: 4px (标签、徽章)

### 3.2 快速录入功能 ⭐核心功能

#### 功能描述
右下角悬浮按钮,点击弹出快速录入表单,可一次性录入体温、用药、症状三项信息,提高录入效率。

#### 页面结构
```
┌─────────────────────────────┐
│  快速录入弹窗                │
│  ┌───────────────────────┐  │
│  │ 快速录入          ✕   │  │
│  ├───────────────────────┤  │
│  │ 🌡️ 体温记录            │  │
│  │ 体温: [____]℃         │  │
│  │ 测量部位:              │  │
│  │ [腋下] [口腔] [耳温] [肛温] │  │
│  ├───────────────────────┤  │
│  │ 💊 用药记录            │  │
│  │ 药品名称: [________]   │  │
│  │ 剂量: [____]          │  │
│  ├───────────────────────┤  │
│  │ 📝 症状记录            │  │
│  │ 症状类型:              │  │
│  │ [咳嗽] [流涕] [呕吐]   │  │
│  │ [腹泻] [皮疹] [发热]   │  │
│  │ 详细描述:              │  │
│  │ [________________]     │  │
│  │ [________________]     │  │
│  ├───────────────────────┤  │
│  │   [保存记录]           │  │
│  └───────────────────────┘  │
└─────────────────────────────┘
```

#### 设计要点
- **悬浮按钮**: 右下角固定位置,圆形,背景色随主题变化
- **弹窗动画**: 从底部滑入,背景半透明遮罩
- **表单设计**: 分为三个独立区块,可选择性填写
- **智能联动**: 录入体温后自动切换主题色
- **标签选择**: 测量部位单选,症状类型多选
- **验证提示**: 至少填写一项才能提交

#### 交互逻辑
1. 点击右下角悬浮按钮打开快速录入弹窗
2. 可选择性填写体温、用药、症状任意一项或多项
3. 录入体温后自动判断健康状态并切换主题
4. 点击保存后关闭弹窗并显示成功提示
5. 点击背景或关闭按钮关闭弹窗

## 十一、动态主题实现说明

### 11.1 技术实现
使用CSS变量和JavaScript实现动态主题切换:

```css
/* 定义主题变量 */
.theme-healthy {
    --theme-primary: #7ED321;
    --theme-secondary: #9BE34D;
    --theme-gradient-start: #7ED321;
    --theme-gradient-end: #9BE34D;
}

.theme-fever {
    --theme-primary: #D0021B;
    --theme-secondary: #FF4D6A;
    --theme-gradient-start: #D0021B;
    --theme-gradient-end: #FF4D6A;
}

.theme-low-fever {
    --theme-primary: #F5A623;
    --theme-secondary: #FFB84D;
    --theme-gradient-start: #F5A623;
    --theme-gradient-end: #FFB84D;
}
```

### 11.2 切换逻辑
```javascript
// 根据体温判断健康状态
function getHealthStatus(temperature) {
    if (temperature < 37.3) {
        return 'healthy';
    } else if (temperature < 38.5) {
        return 'low-fever';
    } else {
        return 'fever';
    }
}

// 应用主题
function applyTheme(status) {
    const appContainer = document.querySelector('.app-container');
    appContainer.classList.remove('theme-healthy', 'theme-fever', 'theme-low-fever');
    appContainer.classList.add(`theme-${status}`);
}
```

### 11.3 应用场景
1. **儿童档案卡片**: 背景渐变色随主题变化
2. **添加按钮**: 背景色随主题变化
3. **状态指示器**: 显示当前健康状态(🟢/🟡/🔴)
4. **体温显示**: 数值颜色根据体温状态变化

### 11.4 用户体验优势
- **直观反馈**: 家长一眼就能看出孩子当前的健康状态
- **情感化设计**: 不同颜色传达不同的情感(放心/关注/警示)
- **减少焦虑**: 健康状态用绿色,让家长安心
- **及时提醒**: 发热状态用红色,提醒家长注意
